# UI Principles

The application should feel like a professional productivity tool.

Avoid flashy colors.

Use a clean dashboard inspired by

- GitHub
- Notion
- Linear
- Raycast

Design principles

- Minimalistic
- Spacious layouts
- Soft shadows
- Consistent spacing
- Rounded corners
- Responsive design

Use Bootstrap 5 together with custom CSS.

Icons should come from Bootstrap Icons.

The dashboard should prioritize today's review queue.

The interface should not resemble AI-generated dashboards.

Design should be inspired by modern SaaS applications.
