# Vision

The purpose of this project is to become a personal learning companion for studying algorithms and data structures.

Rather than focusing on solving problems online, the platform focuses on retaining knowledge.

Every solved problem becomes a knowledge asset that can be reviewed repeatedly through spaced repetition.

The application should encourage active recall before revealing the stored solution.

The user should always know:

- What to study today
- What has already been mastered
- What is likely to be forgotten soon
- Which topics require more attention

The application is designed primarily for a single user.
