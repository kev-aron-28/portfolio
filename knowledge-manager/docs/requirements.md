# Functional Requirements

## Topics

The user can

- Create topics
- Edit topics
- Delete topics
- Browse topics

Examples

- Arrays
- Trees
- Dynamic Programming
- Graphs

---

## Problems

The user can

- Create problems
- Assign a topic
- Assign multiple tags
- Store difficulty
- Store external links
- Mark favorites
- Archive problems

---

## Solutions

Each problem can contain

- Source code
- Explanation
- Complexity analysis
- Personal observations
- Common mistakes

---

## Reviews

Each review stores

- Review date
- Difficulty rating
- Time spent
- Personal notes

---

## Dashboard

Display

- Reviews due today
- Upcoming reviews
- Recently completed reviews
- Learning statistics
- Topic progress
- Heatmap
