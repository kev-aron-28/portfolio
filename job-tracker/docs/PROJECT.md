# Job Tracker

Job Tracker is a local-first application designed to automate job searching and manage the entire job application lifecycle.

## Goals

- Automatically collect job postings from multiple platforms
- Centralize job listings in one place
- Filter jobs based on user-defined profiles
- Track applications and interviews
- Generate analytics and insights

The application uses server-side rendered views using Thymeleaf.

No SPA framework is required.

The UI is generated directly from Spring Boot templates.

## Scope
This is a personal productivity tool focused on job search optimization, not mass application automation.

## Long-term vision

- Job scoring system
- Salary analytics
- Multi-platform scraping engine
