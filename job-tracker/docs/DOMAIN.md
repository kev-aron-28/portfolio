# Domain Model

## Core entities

### Job
Represents a job posting collected from external sources.

### Company
Represents an employer organization.

### Application
Represents a user's interaction with a job posting.

### SearchProfile
Defines filtering criteria for job discovery.

### Interview
Represents a hiring process stage.

### Resume
Represents a versioned CV.

## Rules

- Domain objects are framework-agnostic
- No database annotations
- No external dependencies
