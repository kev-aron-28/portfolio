# Allowed Technologies

Only use the following stack:

## Backend
- Java 21
- Spring Boot 3

## Scraping
- Playwright (preferred)
- Jsoup (HTML parsing only)
- HttpClient (network requests)

## Database
- Postgresql
- Flyway

## Testing
- JUnit 5
- Mockito

## Frontend (Server-side rendering)
- Thymeleaf
- HTML5

## Forbidden
- Selenium (do not use)
- Raw JDBC (except migrations)
- Field injection in Spring
